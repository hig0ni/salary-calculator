# 연봉 실수령액 계산기

세전 연봉에서 4대보험과 소득세·지방소득세를 빼고 **월 실수령액**을 추정한다.

- 라우트: `/` (홈)
- 로직: `lib/salary.js` · UI: `components/SalaryCalculator.js` · 페이지: `app/page.js`
- 상위 규칙: [../CLAUDE.md](../CLAUDE.md) · 공통 인프라: [architecture.md](architecture.md)

---

## 1. 입력

| 상태 | 의미 | 단위/기본값 |
|---|---|---|
| `연봉만원` | 세전 연봉 | 만원 단위 (기본 3600 = 3,600만원) |
| `부양가족수` | 본인 포함 공제대상 가족 수 | 명 (기본 1) |
| `비과세월액` | 월 비과세 급여(식대 등) | 원 (기본 200,000) |

`SalaryCalculator`가 `연봉만원 × 10000`으로 환산해 `calcSalary(연봉, 부양가족수, 비과세월액)` 호출.

## 2. 계산 흐름 — `calcSalary()`

```
월급        = 연봉 / 12
과세월급    = max(0, 월급 − 비과세월액)          # 보험료·세금 부과 기준

# 4대보험 (근로자 부담분, 원 단위 10원 절사)
국민연금 = clamp(과세월급, 하한, 상한) × 4.5%
건강보험 = 과세월급 × 3.545%
장기요양 = 건강보험 × 12.95%
고용보험 = 과세월급 × 0.9%

# 소득세 (연 추정 → 월)
연간과세급여 = 과세월급 × 12
과세표준 = 연간과세급여 − 근로소득공제 − 1,500,000 × 부양가족수 − 연간보험료
산출세액 = 누진세율(과세표준)
결정세액 = 산출세액 − 근로소득세액공제(한도 적용)
소득세   = 결정세액 / 12
지방소득세 = 소득세 × 10%

공제합계   = 4대보험 + 소득세 + 지방소득세
월실수령액 = 월급 − 공제합계
```

반환: `{ 연봉, 월급, 공제:{국민연금,건강보험,장기요양,고용보험,소득세,지방소득세},
공제합계, 월실수령액, 연실수령액 }`.

## 3. 요율·세율 상수 (갱신 포인트) — **매년 점검**

> **모든 값은 `lib/salary.js` 한 파일에만 있음.** 아래 표의 "위치"만 고치면 계산
> 로직은 그대로 재사용된다. 값을 바꾼 뒤에는 **연도 표기도 함께 갱신**(§5 참고).

기준: **2026년** (2026-07 기준으로 작성).

| 항목 | 위치(`lib/salary.js`) | 현재 값 | 출처 / 확인처 |
|---|---|---|---|
| 국민연금 요율 | `RATES.국민연금` | 4.5% | 국민연금공단 (근로자 부담분) |
| 건강보험 요율 | `RATES.건강보험` | 3.545% | 국민건강보험공단 ※2026 확정요율 확인 |
| 장기요양 요율 | `RATES.장기요양` | 건강보험료의 12.95% | 국민건강보험공단 ※2026 확정요율 확인 |
| 고용보험 요율 | `RATES.고용보험` | 0.9% | 고용노동부 (근로자 부담분) |
| 연금 기준소득월액 상·하한 | `연금_상한월소득` / `연금_하한월소득` | 6,370,000 / 400,000 | 국민연금공단, **매년 7월 갱신** |
| 근로소득공제 구간 | `근로소득공제()` | 한도 2,000만원 | 소득세법 §47 |
| 소득세 누진세율 구간·누진공제 | `산출세액()` | 6%~45% (8구간) | 소득세법 §55 |
| 근로소득 세액공제 | `근로소득세액공제()` | 55%/30%, 총급여별 한도 | 소득세법 §59 |
| 인적공제 | `calcSalary()` 내 `1,500,000 × 부양가족수` | 1인당 150만원 | 소득세법 §50 |

### ⚠️ 2026년 갱신 시 확인이 필요한 값

| 값 | 상태 | 비고 |
|---|---|---|
| 건강보험료율 | **직전(2025) 확정값 3.545% 유지** | 2026년 요율 발표 시 반드시 교체 |
| 장기요양보험료율(건보료 대비) | **직전(2025) 확정값 12.95% 유지** | 2026년 요율 발표 시 반드시 교체 |
| 국민연금 상·하한 | 2025.7~2026.6 적용값 반영 | 2026.7에 다시 갱신 |
| 국민연금 요율 | 4.5% (변동 논의 있음) | 연금개혁 확정 시 교체 |

> 위 "확인 필요" 항목은 코드 주석에도 `※2026 확정요율 확인`으로 표시돼 있다.

### 연도 표기를 바꿔야 하는 위치 (전역)
연도(예: 2026)가 문자열로 박혀 있는 곳 — 요율 갱신 시 함께 수정:
- `lib/site.js` → `SITE.title` ("… 2025 …")
- `app/retirement/page.js`, `app/age/page.js` → `metadata.title` / `openGraph.title`
- `app/page.js` → FAQ의 "(2026년 기준 …)"
- `components/SalaryCalculator.js` → `.disclaimer` "2026년 요율 기준"
- `lib/salary.js` → 상단 주석 "기준: 2026년"
> 빠르게 찾으려면 프로젝트에서 직전 연도 문자열(예: `2026`)로 전체 검색.

## 4. 한계 / 주의

- 소득세는 국세청 **간이세액표가 아닌 누진세율 기반 추정**이라, 실제
  원천징수액·연말정산 결과와 수천~수만 원 차이가 날 수 있다.
- 비과세는 월 비과세액만 반영(식대 등). 각종 특별공제·세액공제(연금저축, 의료비
  등)는 미반영.
- 회사별 수당·상여·공제 구조는 반영하지 않는다.
- 하단 `.disclaimer`에 위 내용을 명시(제거 금지).

## 5. 고도화 아이디어

- 월급 실수령 외 **연봉 역산기**(원하는 실수령 → 필요한 세전 연봉).
- 연봉 구간별 실수령액 **표/그래프** 추가(SEO 콘텐츠 강화).
- 비과세 항목 세분화(식대/자가운전/보육수당) 입력.
- 2인 이상 맞벌이·연말정산 시뮬레이션(범위 확장 시 별도 페이지 권장).

# 만 나이 계산기

생년월일로 **만 나이 통일법(2023.6.28 시행)** 기준 만 나이·연 나이와 부가 정보를
계산한다.

- 라우트: `/age/`
- 로직: `lib/age.js` · UI: `components/AgeCalculator.js` · 페이지: `app/age/page.js`
- 상위 규칙: [../CLAUDE.md](../CLAUDE.md) · 공통 인프라: [architecture.md](architecture.md)

---

## 1. 입력

| 상태(UI) | 인자(`calcAge`) | 의미 | 형식/기본 |
|---|---|---|---|
| `생년월일` | `birthStr` | 태어난 날 | `YYYY-MM-DD` (기본 1990-01-01) |
| `기준일` | `baseStr` | 나이 기준 날짜 | `YYYY-MM-DD` (기본 = 오늘) |

## 2. "오늘" 처리 규칙 (중요)

정적 export이므로 빌드 시점과 접속 시점이 다르다. `AgeCalculator`는 초기 `기준일`을
빈 문자열로 두고, **마운트 후 `useEffect`에서 `todayStr()`로 세팅**해 hydration
불일치를 피한다. `기준일`이 비어 있는 첫 렌더에서는 결과를 계산하지 않는다.

- `todayStr()`는 화면 표시용 **로컬 오늘**(`YYYY-MM-DD`)을 반환.
- 이후 계산은 `calcAge()` 내부에서 다시 **UTC 파싱**해 일수/차이를 구한다.

## 3. 계산 흐름 — `calcAge()`

```
만나이 = 기준.연 − 출생.연  (그해 생일 안 지났으면 −1)
연나이 = 기준.연 − 출생.연
총일수 = floor((기준 − 출생) / 하루)               # 태어난 지 N일째
생일까지 = 다음 생일 − 기준 (일)                    # 0이면 오늘이 생일
띠     = 12지 [(출생연 − 4) mod 12]                 # 2020년=쥐 기준
별자리 = 양력 월/일 → 별자리표
태어난요일 = 출생일의 요일
```

반환: `{ valid, 만나이, 연나이, 총일수, 생일까지, 띠, 별자리, 태어난요일 }`.
`valid: false` — 날짜 미입력·형식 오류·기준일 < 생일.

## 4. 세부 규칙

- `parseDate()`가 **UTC 파싱 + 유효성 검사**(2월 30일 등 잘못된 날짜 거부).
- **띠**: `띠목록[(연도 − 4) mod 12]`. 2020년 → `(2020−4)%12=0` → 쥐. 검증됨.
- **별자리**: `별자리표`의 (월, 일 상한)으로 판정. 경계값은 표의 `<=` 규칙에 따름.
- **요일**: `getUTCDay()` → `요일목록`.
- 화면에는 만 나이를 강조(`.result-hero`)하고, 표에 연 나이·총일수·D-day·띠·별자리·
  요일을 나열.

## 5. 법적 배경 (콘텐츠/문구 근거)

- 2023.6.28 시행 **만 나이 통일법**: 법령·계약·문서상 나이는 별도 규정이 없으면
  만 나이 사용.
- **연 나이**(현재 연도 − 출생 연도)는 병역법·청소년보호법 등 일부에서 사용.
- 세는 나이(한국식)는 법적 효력 없음.
- 위 내용은 페이지 FAQ·`.disclaimer`에 반영되어 있음(문구 유지).

## 6. 갱신 포인트

- 계산식은 안정적(법 개정 없으면 변경 불필요).
- 연도 표기(페이지 `metadata`의 "2026")만 정기 갱신.

## 7. 고도화 아이디어

- **디데이/기념일 계산**(1,000일·10,000일, 100세까지 남은 날 등).
- 음력 생일 → 양력 변환(라이브러리 필요, 정적 export 호환 확인).
- 정년·국민연금 수급 개시 나이 등 **나이 기반 파생 계산** 연계.
- 두 날짜 사이 만 나이/개월 수 계산(범용 날짜 계산기로 확장).

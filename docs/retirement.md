# 퇴직금 계산기

입사일·퇴사일과 퇴직 전 3개월 급여로 **근로기준법상 법정 퇴직금**을 추정한다.

- 라우트: `/retirement/`
- 로직: `lib/retirement.js` · UI: `components/RetirementCalculator.js` ·
  페이지: `app/retirement/page.js`
- 상위 규칙: [../CLAUDE.md](../CLAUDE.md) · 공통 인프라: [architecture.md](architecture.md)

---

## 1. 입력

| 상태(UI) | 인자(`calcRetirement`) | 의미 | 단위/기본 |
|---|---|---|---|
| `입사일` | `입사일` | 입사 날짜 | `YYYY-MM-DD` |
| `퇴사일` | `퇴사일` | 마지막 근무 **다음 날**(=퇴직일) | `YYYY-MM-DD` |
| `월급여만원` | `월급여` | 퇴직 전 3개월 **월평균** 세전 급여 | 만원→원 |
| `연간상여만원` | `연간상여금` | 1년간 상여금 총액(선택) | 만원→원 |
| `연차수당만원` | `연차수당` | 1년간 연차수당 총액(선택) | 만원→원 |

UI가 만원 단위를 `× 10000`으로 환산해 객체로 전달.

## 2. 계산 흐름 — `calcRetirement()`

```
재직일수  = 퇴사일 − 입사일 (일)
기간      = 년/월/일 분해 (breakdownPeriod)

# 퇴직 전 3개월 (달력 기준, 보통 89~92일)
기간일수  = 퇴사일 − (퇴사일의 3개월 전 같은 날)

# 3개월 임금 총액
임금총액  = 월급여×3 + 연간상여금×(3/12) + 연차수당×(3/12)

1일 평균임금 = 임금총액 / 기간일수
법정 퇴직금  = round( 1일평균임금 × 30 × 재직일수 / 365 )
```

반환: `{ valid, 재직일수, 기간:{년,개월,일}, 지급대상, 일평균임금, 임금총액,
기간일수, 상여가산, 연차가산, 퇴직금 }`.

- `valid: false` — 날짜 미입력/역전(퇴사일 ≤ 입사일) 시. UI가 안내 문구 표시.
- `지급대상: false` — 재직 **365일 미만**. 법정 퇴직금 대상 아님을 `.notice`로 경고
  (금액은 참고 환산값으로 계속 표시).

## 3. 날짜 처리 규칙 (중요)

- 모든 날짜는 `parseDate()`로 **UTC**(`Date.UTC`) 파싱 → 시간대 오차 제거.
- 일수 차이는 `daysBetween()`(밀리초/86,400,000, 반올림).
- "3개월 전"은 `getUTCMonth() - 3`으로 달 경계 자동 처리(월별 일수 차이 반영).
- `breakdownPeriod()`는 일→월→년 순으로 빌림 처리하여 년·월·일 표기.

## 4. 산정 근거 / 한계

- 근거: 근로기준법상 **평균임금 × 30일 × (재직일수/365)**.
- 평균임금 = 퇴직 전 3개월 임금총액 ÷ 그 기간의 총 일수. 상여·연차수당은 연액의
  3/12를 가산.
- **미반영/한계**
  - 평균임금이 **통상임금보다 적으면 통상임금** 적용(근로기준법 제2조) — 본
    계산기는 입력한 3개월 급여 기준 추정만 수행.
  - **퇴직연금(DC형)**은 회사 납입액+운용수익으로 결정되어 이 공식과 다르다.
  - 주 15시간 미만·1년 미만 등 지급요건은 `지급대상` 플래그로만 안내.
- 하단 `.disclaimer`에 위 내용을 명시(제거 금지).

## 5. 갱신 포인트

- 공식·요율성 상수는 없음(법정 산식 고정). 다만 **통상임금 판례** 변화가 크면
  안내 문구·계산 방식 재검토.
- 연도 표기(페이지 `metadata`의 "2026")는 정기 갱신.

## 6. 고도화 아이디어

- **통상임금 입력**을 추가해 평균임금과 비교, 큰 값 자동 적용.
- 3개월 급여를 **월별 개별 입력**(달마다 다른 급여)으로 정밀화.
- 퇴직소득세 추정 추가(실수령 퇴직금).
- 퇴직연금(DC/DB) 안내 분기.
